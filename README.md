# ¡Hola! Soy Alejo Nigrelli 👋
### Full Stack Web & Mobile Developer
 
Soy un desarrollador Full Stack especializado en la creación de aplicaciones web y móviles de punta a punta: desde el diseño de la base de datos y la seguridad de los datos, hasta la interfaz que el usuario final toca. Tengo experiencia construyendo arquitecturas completas con React Native, PostgreSQL y Supabase, además de un costado en Python orientado a APIs y análisis de datos, siempre con foco en entender el problema de negocio antes de escribir la primera línea de código.
